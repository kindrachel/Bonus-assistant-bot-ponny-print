# Реферальный бот для Типографии "PonnyPrint"

**Бот выполняет стандартные функции реферального бота:** создает индвидуальный код каждому пользователю, вставляет его в заготовленную ссылку и через нее бот понимает кем был пригоашен участник

Коды сохраняются в базу данных. База данных SQLite, бибилиотека Aiogram версия Python 3.12.2

# .env значения 

BOT_TOKEN= Ваш токен из http://t.me/botfather
URL_DATABASE=Ссылка на sqlite

# Запуск бота

1. pip install -r requirements.txt
2. pip install dotenv
3. nano .env (создает файл .env) --> вписываете значения ^^ как в пункте выше
4. python main.py

**Если все верно то бот запуститься**
